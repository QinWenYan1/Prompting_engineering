# 📘 18 结语 (Conclusion)

> 来源：Andrew Ng | Module 3: Working with Multimedia & Code | 结课 | ~2 分钟

---

## 🧠 本课核心

从 AI 新手到 AI 高手的旅程终点——也是一段新旅程的起点。你已经掌握了整套 prompt 工程的工具箱，现在唯一要做的就是：**去用**。

---

## ✅ Andrew Ng 的结课寄语

> "With all the techniques you've learned, you're now ready to be an AI power user."
> 「掌握了所有这些技巧，你现在已经准备好成为一名 AI 高手了。」

- **回顾你学会的能力**

    | 模块 | 核心能力 |
    |------|---------|
    | Module 1: 信息获取 | 区分预训练知识、网络搜索、深度研究——用对工具获取准确信息 |
    | Module 2: AI 作为思想伙伴 | 头脑风暴、上下文窗口、推理引擎、破解谄媚、渐进式写作、量规审阅 |
    | Module 3: 多媒体与代码 | 图像理解与生成、语音克隆、构建应用、数据分析 |

- **你现在可以做的事**
    - 用 AI 头脑风暴创意方案
    - 用 AI 撰写和审阅报告
    - 用 AI 处理图像、语音、视频
    - 用 AI 写代码分析数据
    - 用 AI 构建可用的软件应用

---

## 📌 三个最终建议

1. **保持迭代**——prompt 工程不是一次写对，而是反复改进。第一版 prompt 几乎从来不是最好的。
2. **保持尝试**——AI 模型每个月都在变强。上个月不可能的事，这个月可能就行。最坏的情况就是没成功——然后你就知道了边界在哪。
3. **保持负责**——AI 是强大的工具，用于有益的目的。语音克隆可以修播客，也可以搞诈骗。选择在你自己。
