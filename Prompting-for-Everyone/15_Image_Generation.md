## ✅ 知识点1: 让 AI 帮你写图像 Prompt

**图像生成不只是「输入一句话得到一张图」**



- **核心技巧**：文字 AI 比你会写图像 prompt——它知道图像生成模型需要什么描述格式

- **直接想到的 prompt（新手）**
    ```
    一只猫在晚上开咖啡馆
    ```

- **让文字 AI 帮你扩展（高手）**
    ```
    生成一个图像 prompt：一只猫在夜间秘密经营一家咖啡馆。
    generate a prompt for an image of a cat secretly running a coffee shop at night.
    ```
    → AI 输出的 prompt 包含：
    - 场景设定（setting）
    - 角色细节（details of the character）
    - 氛围或风格（a mood or a style）

- **你可以根据以上特点来自由调整提示词**

<a id="id2"></a>
## ✅ 知识点2: 掌握视觉艺术词汇——这是专家的秘密武器

**可以用视觉艺术词汇来描述风格**

- 图像生成的核心技能不是「会写 prompt」，而是**有词汇量来描述你想要的视觉风格**

- **为什么艺术背景的人生成图像更厉害**：艺术爱好者和艺术史爱好者擅长图像 prompt 工程——因为他们有精准的描述语言

    | 普通描述 | 精准的艺术描述 |
    |---------|-------------|
    | 「好看的风格」 | `cinematic（电影感）`、`watercolor（水彩）`、`cyberpunk（赛博朋克）`、`anime（动漫风）` |
    | 「光线柔和」 | `golden hour lighting（黄金时刻光）`、`chiaroscuro（明暗对照法）` |
    | 「画面很有感觉」 | `ethereal（空灵的）`、`gritty（粗粝的）`、`surreal（超现实的）` |

- **快速学习法**
    1. 找一张你喜欢的 AI 生成图
    2. 上传给 AI，问「你会如何描述这张图？」
    3. 看 AI 用了什么词汇——**学会那些词，下次自己用**

- **案例：老照片修复**
    - 上传一张褪色的童年照片（Andrew 和弟弟、朋友）
    - Prompt: `remove the glare and the rough texture, and make it a more natural aspect ratio（去掉反光和粗糙纹理，调成更自然的比例）`
    - 用 Google NanoBanana 模型完成修复

<a id="id3"></a>
## ✅ 知识点3: 扩散模型怎么工作——和文本生成完全不同

**文本生成 vs 图像生成**

| | 文本生成 | 图像生成（扩散模型） |
|------|---------|-----------------|
| 输出方式 | **逐词（piece by piece）** 生成 | **整张图一次**生成 |
| 训练过程 | 预测下一个 token | 从纯噪声中**逐步去噪** |
| 推理过程 | 可以中途停止 | 必须全部完成 |
| 耗时 | 数秒 | 数十秒 |
| 成本 | 不到一美分 | 数美分 |

**扩散模型的工作原理**
```
纯噪声（随机像素值）
    ↓ 逐步减去噪声
    ↓ 每一步都让图像更清晰一点
    ↓
最终清晰图像
```

训练时，模型学习如何从加了噪声的图像中恢复出原始图像。推理时，从一个完全随机的噪声出发，逐步「去噪」直到生成清晰的结果。

**注意点**
- 💡 理解扩散模型有助于调试：如果你生成的图有问题（比如手指多了一根），这不是 prompt 的问题，是模型在去噪过程中「猜」错了

<a id="id4"></a>
## ✅ 知识点4: AI 图像的三种常见翻车

| 错误类型 | 表现 | 现代模型（NanoBanana）的改进 |
|---------|------|--------------------------|
| **诡异的手** | 手指比 5 根多或少、关节变形 | 大幅改善 |
| **乱码文字** | 「happy birthday」拼写得一塌糊涂 | 开始能生成可读的文字 |
| **角色不一致** | 同一角色的发型/服装在两张图之间变了 | 能做到「相当一致的角色」 |

**Andrew Ng 关于成本的说明**
> "生成任何 AI 内容的成本都在持续下降。"——今天贵的东西，明天可能就便宜了。
